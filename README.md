# Emotion-diary



- 설명 : 내 감정을 아주 좋음에서 아주 나쁨까지 표시하고 제목과 게시글을 작성하여 리스트 형식으로 한눈에 볼 수 있는 일기장 <br />
- 기획 및 제작 : 최진주 <br />
- 제작기간 : 2023.10.02 ~ 2024.01.04 <br />
